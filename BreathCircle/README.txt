============================================
BreathCircle (呼吸圆圈) V1.0
============================================

BreathCircle 是一款 Windows 桌面呼吸训练工具。
屏幕显示一个透明玻璃质感的彩色圆形，
通过圆形的缩放动画引导用户进行呼吸练习。

===== 系统要求 =====
- Windows 10 / 11 x64
- .NET 8.0 运行时（单文件发布则无需安装）

===== 安装与运行 =====
无需安装，双击 BreathCircle.exe 即可运行。
应用启动后会在屏幕中央显示一个彩色圆形。

发布命令：
  dotnet publish -c Release -r win-x64 -p:PublishSingleFile=true -p:SelfContained=true

===== 基本操作 =====

  单击圆形     开始呼吸 / 暂停 / 继续
  拖拽圆形     移动窗口位置
  右键圆形     打开功能菜单
  Ctrl+Alt+R   开始 / 暂停
  Ctrl+Alt+N   切换下一个模式
  Ctrl+Alt+W   进入 Wim Hof 模式
  Ctrl+Alt+H   显示 / 隐藏窗口
  Ctrl+Alt+Q   退出程序

  暂停与停止的区别：
  - 单击 = 暂停（保留当前进度，再单击继续）
  - 右键菜单「停止」= 完全重置回到空闲

===== 呼吸模式 =====

1. 盒式呼吸 (Box Breathing)
   吸4秒 → 屏4秒 → 呼4秒 → 屏4秒 → 循环

2. 4-7-8 呼吸法
   吸4秒 → 屏7秒 → 呼8秒 → 循环

3. 共鸣呼吸 (Resonance Breathing)
   吸5秒 → 呼5秒 → 循环（无屏息）

4. 生理性叹息 (Physiological Sigh)
   吸2秒 → 屏2秒 → 呼8秒 → 循环

5. Wim Hof 呼吸法（参数可调，见下文）
   默认每轮：30次快速呼吸 → 屏息 → 深吸气+屏住 → 缓慢吐气 → 下一轮

===== Wim Hof 设置 =====

右键菜单「Wim Hof 设置」可调整以下参数：

  练习轮次        1~10 轮（默认 4）
  每轮呼吸次数    1~100 次（默认 30）
  每次吸气时长    0.3~5.0 秒（默认 1.0）
  每次呼气时长    0.3~5.0 秒（默认 1.0）
  第1轮屏息时长   10~300 秒（默认 30）
  每轮屏息递增    0~120 秒/轮（默认 15）
                 例：30 → 45 → 60 → 75
  恢复屏息时长    5~60 秒（默认 15）
  恢复吐气缓冲    1~30 秒（默认 3）

  屏息阶段显示目标倒计时，到达目标自动进入下一阶段。
  安全上限 = 最大屏息 + 60 秒缓冲。

===== 颜色设置 =====

右键菜单「颜色设置」可为每种呼吸模式自定义颜色：

  - 左侧列表选择模式
  - R / G / B 滑块拖拽调色
  - 也可直接输入 6 位十六进制色码
  - 色块实时预览
  - 「全部默认」一键恢复系统默认配色

颜色配置保存在 Config/settings.json 中。

===== 显示选项 =====

右键菜单 / 系统托盘菜单：

  显示阶段文字    ON: 显示"吸气""屏息"等文字
                 OFF: 隐藏阶段文字

  显示倒计时      ON: 显示"4s""3s"等秒数
                 OFF: 去掉倒计时数字

  二者可独立开关：

    阶段文字 + 倒计时 → "屏息 4s"
    仅阶段文字         → "屏息"
    仅倒计时           → "4s"
    全部关闭           → (无文字)

===== 系统托盘 =====

- 关闭窗口时自动最小化到系统托盘
- 右键托盘图标：模式切换、显示选项、退出
- 双击托盘图标：显示窗口

===== 窗口功能 =====

  置顶          窗口始终在最前
  点击穿透      鼠标事件透传（穿透模式下需通过快捷键/托盘操作）

===== Wim Hof 安全提示 =====

Wim Hof 呼吸法包含强力呼吸和长时间屏息，
可能导致头晕、四肢麻木或意识模糊。

请勿在以下情况练习：
- 驾驶或操作机械时
- 游泳或靠近水域时
- 患有心脏病、高血压或癫痫
- 怀孕期间

首次启动 Wim Hof 时会显示安全确认对话框。

===== 数据存储 =====

设置文件：Config/settings.json
保存内容：当前模式、窗口位置、置顶/穿透状态、
         显示选项、Wim Hof 参数、自定义颜色。

===== 技术信息 =====

- 框架：WPF + .NET 8 + MVVM (CommunityToolkit.Mvvm)
- 动画：CompositionTarget.Rendering (60FPS)
- 缓动：EaseInOutSine
- 计时：Stopwatch 真实时间（无帧率漂移）
- 状态：State Pattern
- 视觉：玻璃拟态渐变 + 圆环描边 + 模式专属配色
